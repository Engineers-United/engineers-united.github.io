    How to get involved

    I’m new… now what?

    1. Read the documentation on your role in the HackUNT/roles/ section
    2. Coordinate with the president about what is needed from you
    3. Attend the meetings
    4. Have fun, and enjoy the experience!

    Volunteering at HackUNT

    In our opinion, volunteering for HackUNT is the best way to earn Volunteering hours. We are a student-run, non-profit organization.

    Volunteers have the ability to earn up to 36 Volunteering hours. In addition, Volunteers will get free food, drinks, snacks, and their very own HackUNT themed Volunteering T-shirt.

    Start by reading the Volunteer Guidelines

    Aspiring Engineer

    1. Read all of the Git Workflows documentation
    2. Read the entire README.md file of the repository you wish to contribute too
    3. Select an issue or create a feature you would like to work on
    4. Create a new branch off of the beta branch
    5. Happy Hacking!

    HackUNT Checklist

    This checklist is everything I can remember that needs to be done to make HackUNT possible.
    This list is not in order it is just everything I can remember and will be added to as needed.

    • Set up Typeform for Registration
    • Pick a theme for the event
    • Make Sponsor Packet ( Use Canva )
    • Make Sponsor Onboarding Packet
    • Pick a date for the event
      • Set up the event on Org Sync ( you will need to keep up with this and answer their comments. )

      • Create a new design for website
      • Create a Typeform on the website for registration

      • Stock toiletries in the bathroom
    • Set up Wifi for the Event
    • Set up Parking for the Event

    • MLH ( At first I thought why then they made up for it at the event! )
